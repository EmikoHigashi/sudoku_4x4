# 4x4 ナンプレメーカー (Kids Sudoku Maker)
https://emikohigashi.github.io/sudoku_4x4/

お子様が鉛筆で書き込みやすい「大きなマス目」の4x4ナンプレを、ブラウザで気軽に生成・印刷できるツールです。
Pythonのロジックをベースに、GitHub Pagesで誰でも手軽に使えるWebアプリとして公開しています。

## ✨ このツールの特徴
- **お子様向けデザイン**: 数字やマス目が大きく、小さな手でも書き込みやすい設計です。
- **A4印刷に最適化**: 「印刷」ボタンを押すと、1ページにパズルがちょうど収まるようにレイアウトされます。
- **難易度調整**: 空欄（穴）の数を自由に変更できるので、初めてのナンプレから少し難しいものまで調整可能です。
- **インストール不要**: ブラウザだけで動作し、スマホやタブレットからも操作できます。

## 🚀 使い方
1. **設定**: 「生成する数」と「空欄の数（難易度）」を入力します。
2. **生成**: 「パズルを新しく作る」ボタンを押すと、ランダムに問題が作成されます。
3. **印刷/保存**: 「A4 PDF保存 / 印刷」ボタンを押し、プリンター設定で「PDFに保存」またはご家庭のプリンターを選択してください。

## 🛠 技術仕様
- **ロジック**: Python (`generate_4x4_sudoku.py`) のアルゴリズムをJavaScriptに移植。
- **UI**: HTML5 / CSS3 (Print Media Queries)
- **ホスティング**: GitHub Pages
